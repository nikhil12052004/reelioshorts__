# 🎥 ReelioShorts

ReelioShorts is a modern short-video sharing platform inspired by YouTube Shorts. Users can sign in, upload videos, browse a video feed, and manage their profiles. The project is built with Next.js App Router and follows a scalable architecture using TypeScript, Prisma, and PostgreSQL.

## ✨ Features

* User authentication with Clerk
* Upload and share short videos
* Browse video feed
* User profile management
* Responsive UI
* Dark mode support
* Prisma ORM with PostgreSQL
* Built with Next.js App Router

## 🛠 Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* shadcn/ui
* Prisma
* PostgreSQL
* Clerk Authentication
* Vercel

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/nikhil12052004/reelioshorts__.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## 🔑 Environment Variables

Create a `.env` file and add:

```env
DATABASE_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```
